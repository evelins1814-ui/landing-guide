# Лендинги под вакансии — инструкция для рекрутеров

Внутренний сайт-инструкция: как собрать лендинг под вакансию в Claude Design.
Пошаговые действия, чек-листы, шаблоны промпта и meta.json, варианты лендингов и файлы для скачивания.

## Структура

```
.
├── index.html        ← сам сайт (одна страница)
├── README.md
└── files/            ← файлы, которые скачивают и грузят в Claude Design
    ├── instrukciya-lendingi-vakansii.md   (инструкция: содержание и дизайн)
    ├── claude-design-landing-guide.md     (техника: форма, meta.json, запреты)
    ├── recruto-brand-kit.md               (бренд Рекруто — для внутренних вакансий)
    ├── recruto-logo-datauri.txt           (лого Рекруто — data-URI)
    ├── prompt-template.md                 (шаблон промпта)
    └── meta.template.json                 (шаблон meta.json)
```

## Как выложить на GitHub Pages

1. Создай репозиторий (например `landing-guide`) и залей в него все файлы с сохранением структуры.
2. Settings → Pages → Source: ветка `main`, папка `/ (root)` → Save.
3. Через минуту сайт будет доступен по адресу `https://<твой-логин>.github.io/<репозиторий>/`.

Кнопки «скачать» работают, потому что файлы лежат в папке `files/` рядом с `index.html`.

## Как обновлять инструкции

Замени нужный файл в папке `files/` и закоммить — ссылки на сайте обновятся автоматически,
менять `index.html` для этого не нужно.
